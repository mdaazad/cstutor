# CSTutor — AI Computer Science Tutor

An AI-powered tutoring web app for computer science students. Ask any CS question and get a clear, educational answer powered by Google Gemini.

## Features

- User signup and login with secure password hashing (bcrypt)
- Session-based authentication — dashboard is protected
- Ask any computer science question (data structures, algorithms, JavaScript, Python, Big O, etc.)
- Answers generated by Google Gemini AI
- Full question/answer history saved to MongoDB
- Delete past questions (full CRUD demonstrated)
- Responsive design

## Tech Stack

- **Backend:** Node.js, Express.js
- **Templating:** EJS
- **Database:** MongoDB Atlas with Mongoose
- **AI API:** Google Gemini (`gemini-3.5-flash`)
- **Auth:** express-session + bcryptjs
- **Deployment:** Render

## How to Run Locally

1. Clone the repository

   ```
   git clone https://github.com/mdaazad/cstutor.git
   cd cstutor
   ```

2. Install dependencies

   ```
   npm install
   ```

3. Create a `.env` file in the root folder with these values:

   ```
   MONGO_URI=your_mongodb_atlas_connection_string
   SESSION_SECRET=any_long_random_string
   GEMINI_API_KEY=your_gemini_api_key
   PORT=3000
   ```

4. Start the server

   ```
   node app.js
   ```

5. Visit `http://localhost:3000`

## Database Schema

**Users collection**
| Field | Type | Description |
|-------|------|-------------|
| \_id | ObjectId | Auto-generated primary key |
| username | String | Display name |
| email | String | Unique, used for login |
| passwordHash | String | bcrypt hash — plain text never stored |
| createdAt | Date | Account creation timestamp |

**Chats collection**
| Field | Type | Description |
|-------|------|-------------|
| \_id | ObjectId | Auto-generated primary key |
| userId | ObjectId | Foreign key referencing Users.\_id |
| question | String | The question the user asked |
| answer | String | Gemini's response |
| createdAt | Date | When the question was asked |

## Environment Variables

| Variable       | Description                            |
| -------------- | -------------------------------------- |
| MONGO_URI      | MongoDB Atlas connection string        |
| SESSION_SECRET | Secret key for signing session cookies |
| GEMINI_API_KEY | Google Gemini API key                  |
| PORT           | Server port (default: 3000)            |
